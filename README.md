# Authentication System - Passport.Js

Authentication System with Nodejs, Express, MongoDB and Passport

## Instalation

Install all dependencies

```
npm install
```

MongoDB is a on-demand database, just run the app.

Run app

```
npm start
```

## Built With

* Node.Js
* Express.Js
* MongoDB
* Passport.Js
* Bootstrap
